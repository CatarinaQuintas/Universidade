{\footnotesize\ttfamily int interpretador (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{struct_e_s_t_a_d_o}{E\+S\+T\+A\+DO}} $\ast$}]{e }\end{DoxyParamCaption})}



Função interpretador. 

Nesta função são utilizados 6 comandos\+: gr, ler e Q, movs, pos,jog, jog02 . O comando gr grava o tabuleiro num ficheiro. O comando ler vai buscar o ficheiro guardado pelo comando gr e assim o jogo tem continuidade. O comando Q termina o jogo. O comando movs apresenta os movimentos atá ao momento. O comando pos utilizado para regressar a uma jogada pretendida. O comando jog faz uma jogada aleatoria. O comando jog02 faz uma jogada usando a heurística euclidiana. \mbox{\Hypertarget{interface_8c_aa0b6c1689e60ac852a73d67e066c1df5}\label{interface_8c_aa0b6c1689e60ac852a73d67e066c1df5}}
